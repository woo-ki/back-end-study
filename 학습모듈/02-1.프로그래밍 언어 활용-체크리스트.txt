프로그래밍 언어 활용

체크리스트형 대비

1. java 코드 작성 문제입니다.

2. 출제유형
    (1) Scanner 입력 및 if 처리문제
    (2) 클래스 생성 및 객체 생성 문제
    (3) 슈퍼클래스와 서브클래스 생성자 및 메소드 호출 문제
    (4) Random 클래스를 활용한 퀴즈 문제
        - 랜덤 생성 방법: 예 1~ 1000 사이 랜덤을 생성하시오.	

3. 전체 완성이 아닌 괄호() 채우기 유형(1,3,4번 유형) + 전체 완성 유형(2번 유형)

4. 이클립스 대신 "메모장" + "명령프롬프트"로 풀이합니다.
    1) C드라이브에 test 폴더를 만든다.
    2) test 폴더에 Test.java 파일을 만든다.
    3) 명령 프롬프트를 실행한다.
        (1) 실행창(로고키 + R) 열고, 명령 cmd 입력한다.
        (2) 명령 cd c:\test 를 입력한다.
        (3) 명령 javac Test.java 를 입력한다. 문제가 없으면 아무 메시지가 나타나지 않는다.
            아무 메시지가 나타나지 않을때까지 수정 + 저장 + 다시 실행한다.
	기존에 입력한 명령은 방향키로 자동 완성할 수 있다.
        (4) 명령 java Test 를 입력한다. 문제가 없으면 실행 결과가 출력된다.
