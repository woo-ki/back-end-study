DBCP 방식을 이용한 CRUD 처리

1. Package Explorer에서 Servers - Tomcat v8.5 - context.xml을 연다.

2. 다음 <Resource> 태그를 추가한다.
    <!-- Oracle XE -->
    <Resource 
		auth="Container" 
		name="jdbc/oracle" 
		type="javax.sql.DataSource" 
		driverClassName="oracle.jdbc.driver.OracleDriver" 
		url="jdbc:oracle:thin:@localhost:1521:xe"
		username="spring"
		password="1111"
		maxTotal="8"
		maxIdle="2"
		maxWaitMillis="5000"
	/>

3. 인코딩을 위해서 web.xml을 열고 아래 태그를 추가한다.

	<!-- 인코딩 -->
	<filter>
		<filter-name>encodingFilter</filter-name>
		<filter-class>org.springframework.web.filter.CharacterEncodingFilter</filter-class>
		<init-param>
			<param-name>encoding</param-name>
			<param-value>UTF-8</param-value>
		</init-param>
		<init-param>
			<param-name>forceEncoding</param-name>
			<param-value>true</param-value>
		</init-param>
	</filter>
	<filter-mapping>
		<filter-name>encodingFilter</filter-name>
		<url-pattern>/*</url-pattern>
	</filter-mapping>
	
4. ojdbc6.jar 를 준비한다. (둘 중 하나)
   1) pom.xml에 디펜던시를 추가한다.
      : 현재 프로젝트에서만 사용하겠습니다.
      
      (1) <dependencies> 위에 <repositories>를 추가합니다.
      	<!-- ojdbc6.jar가 저장된 repository 설정 -->
		<repositories>
			<repository>
				<id>oracle</id>
				<name>ORACLE JDBC Repository</name>
				<url>http://maven.jahia.org/maven2</url>
			</repository>
		</repositories>

      (2) 아래 디펜던시를 추가합니다.
      	<!-- ojdbc6.jar 디펜던시 -->
		<dependency>
			<groupId>com.oracle</groupId>
			<artifactId>ojdbc6</artifactId>
			<version>12.1.0.2</version>
		</dependency>
      
   2) apache-tomcat-8.5.59\lib 디렉토리에 ojdbc6.jar를 복사해 둔다.
      : 현재 톰캣을 사용하는 모든 프로젝트에서 사용하겠습니다.

5. taglib 라이브러리를 준비한다. (둘 중 하나)
   1) pom.xml에 디펜던시를 추가한다.
      : 현재 프로젝트에서만 사용하겠습니다.

    <dependency>
      <groupId>org.apache.taglibs</groupId>
      <artifactId>taglibs-standard-spec</artifactId>
      <version>1.2.5</version>
    </dependency>
    <dependency>
      <groupId>org.apache.taglibs</groupId>
      <artifactId>taglibs-standard-impl</artifactId>
      <version>1.2.5</version>
    </dependency>
   
   2) apache-tomcat-8.5.59\lib 디렉토리에 {taglibs-standard-spec-1.2.5.jar, taglibs-standard-impl-1.2.5.jar}를 복사해 둔다.
      : 현재 톰캣을 사용하는 모든 프로젝트에서 사용하겠습니다.
      
6. 롬복(lombok)을 사용하는 경우 Project lombok 디펜던시를 추가한다.
    <!-- https://mvnrepository.com/artifact/org.projectlombok/lombok -->
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <version>1.18.12</version>
        <scope>provided</scope>
    </dependency>      
      
7. 버전 최신화를 진행한다.   버전최신화 방법대로.
